# 🅰️ Angutrain

Projet d'apprentissage Angular réalisé en 30 jours.

## Stack technique
- Angular 19 (Standalone Components)
- TypeScript
- RxJS
- NgRx
- Angular Signals

| J13-J14 | RxJS, Observables | /blog |
| J15 | Projet mi-parcours | /blog |
